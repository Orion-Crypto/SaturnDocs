Upgrade Project Page

The Upgrade Projects page allows you to create leveling and upgrade systems for your Upgrade Project.

User Interface​

The upgrade project can only be accessed by having an existing NFT Project. You can learn how to create an NFT Project here. When you have an NFT Project that you would like to create an Upgrade Project for. Click on the green button that says "Create Upgrade Project" on the Upgrade Project page.

Upgrade projects can de deleted by deleting the corresponding NFT Project on the NFT Projects page.
