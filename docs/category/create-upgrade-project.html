Create Upgrade Project

This section will guide you through the process of creating a new Upgrade Project.

📄️ Upgrade Project Page

The Upgrade Projects page allows you to create leveling and upgrade systems for your Upgrade Project.

📄️ Upgrade NFTs

The Upgrade NFTs page allows you to create, update, and delete level paths for your NFTs

📄️ Upgrade Levels

The Upgrade Levels page allows you to customize the level requirements and price of upgrading your project's NFTs.

📄️ Image / Metadata Upload

The Upload Images page allows you to bulk upload both images and metadata for your Upgrade Project. This is very similar to tyhe Upload Images page in the NFT Project section.
